Hive is the client side database for saving the users drawings
These are the models being saved inside hive

ArtworkModel - typeId: 0
   String id
   String? serverId
   String title
   String description
   List<StencilModel> stencilList
   List<StrokeModel> strokeList

StencilModel - typeId: 1
   String name
   Unit8List imageData

ImageModel - typeId: 2
  String url;
  int? size;
  String? orig_name;
  String? mime_type;
  bool is_stream;
  dynamic meta;
  Uint8List? content;

StrokeModel - typeId: 3
   List<OffsetModel> points
   int color
   double brushSize

OffsetModel - typeId: 4
   double dx
   double dy
